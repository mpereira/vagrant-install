# vagrant-install

Automatically install VirtualBox and Vagrant.

## Installs
  - VirtualBox 4.3
  - Vagrant 1.3.5
  - vagrant-vbguest

## Compatibility
  - Ubuntu

## Tested on
  - Ubuntu 13.10
